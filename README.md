# 🎙️ TTS Translator

텍스트 읽기, 번역, 단어 사전 기능을 제공하는 웹 앱입니다.

## ✨ 주요 기능

### 📖 원문 읽기
- 입력한 텍스트를 선택한 언어로 읽어줍니다
- 읽기 속도 조절 가능 (0.5x ~ 2.0x)
- 8개 언어 지원: 한국어, 영어(US/UK), 일본어, 중국어, 스페인어, 프랑스어, 독일어

### 🌐 번역하기
- 여러 언어 간 자동 번역
- 자동 언어 감지
- 번역 결과에서 단어 클릭 시 사전 기능

### 🔊 번역 후 읽기
- 텍스트를 번역하고 자동으로 읽어줍니다
- 번역과 음성 출력을 한 번에!

### 📚 단어 사전
- 번역된 텍스트의 단어를 클릭하면 번역 표시
- 모르는 단어를 즉시 확인

## 🚀 사용 방법

### 웹 브라우저
1. [https://soonhak-ahn.github.io/tts-translator-app/](https://soonhak-ahn.github.io/tts-translator-app/) 접속
2. 원하는 기능 선택
3. 텍스트 입력 후 사용

### iPhone/iPad (홈 화면에 추가)
1. Safari로 웹사이트 접속
2. 공유 버튼 탭
3. "홈 화면에 추가" 선택
4. 앱처럼 사용 가능!

## 🛠️ 기술 스택

- **HTML5/CSS3**: 반응형 UI
- **JavaScript**: Web Speech API, Fetch API
- **PWA**: 오프라인 지원, 홈 화면 추가 가능
- **Translation API**: MyMemory (무료 번역 API)

## 📱 지원 플랫폼

- ✅ iOS (Safari)
- ✅ Android (Chrome)
- ✅ Desktop (Chrome, Safari, Firefox, Edge)

## 🎯 주요 특징

- 📱 **모바일 최적화**: 터치 친화적 UI

- 🎨 **아름다운 디자인**: 그라디언트 배경, 부드러운 애니메이션
